旅行的意义

发表于 2018-09-04

稍纵即逝的美，只有在特定的时机，特定的场合，特殊的心情，一切一切的巧合发生，才能捕捉到。如同对的那个人，需要特殊的因子巧合的组合，最终某个交叉路口，相遇，相知，相识，相守。

闭上眼睛，你会想到的美是什么？

是凯恩斯浮潜时候看到的最后的珊瑚礁？
是悉尼海湾18000英尺跳伞，远处的水天一色，俯瞰的碧海沙滩？
是在墨尔本俯瞰的十二门徒？
是在旧金山看九曲花街月落？
是古罗马斗兽场内的壮观设计？
是圣马可广场钟楼的鸽子？
是梵蒂冈城外天使城堡暴雨的下午？
是在巴黎之眼上俯瞰暮色的巴黎？

都不是，浮现脑海的是宁静的夜晚，在格里菲斯天文台山的盘山公路上，俯瞰天使城的星星点点，没有路灯的干扰，只是开车匆匆下山时的一瞥，便终身难忘，如同看见了繁星，一瞬间震撼了心灵。

Go memory model

发表于 2018-09-02

前言：Go内存模型能保证 一个协程A对变量a有修改，则另一个协程B取变量a的值时，协程B能知道协程A对变量a的修改，并获得变量a的最新值，即这是一种原子操作，协程A\B共享一块内存。但是，如果一段程序想修改多个协程公用的变量时，需要将多协程的数据访问变为序列化访问。

Happens Before原则

对于单个协程：对变量v的读操作r，写操作w，满足一下两点，可以保证能读到最新的修改。

  1. 写在读之前或同时发生；在写操作w后不会有另一个写操作w’在读操作r前发生。
  2. 写在读之前发生；在写操作w之前和读操作r之后，不会有任何读写操作。

    对于多个协程：必须通过同步事件，保证读操作能读到想要的结果。
